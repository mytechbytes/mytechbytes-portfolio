QuoteAssist
Personal Project · AI-Powered Quote Processing Platform
End-to-end AI platform that ingests travel quote requests from Outlook, web forms, or any channel — extracts structured data, enforces corporate policies, generates pricing-ready JSON, and drafts intelligent email responses. Designed to self-improve through agent feedback, progressively graduating patterns to auto-send.
Elixir Phoenix LiveView OTP Python LLM / GenAI RAG React MS Graph API Outlook Add-in Multi-tenant PostgreSQL REST APIs
Core Features
Multi-channel Ingestion
Outlook Add-in, web forms, and extensible webhook sources — all normalised through a unified ingestion pipeline before AI processing.
AI-Driven Quote Extraction
LLM parses free-form text to extract itineraries, services, dates, passenger counts — each field scored with a confidence value. Low-confidence fields flagged for human review.
Corporate Policy Enforcement
Validates against configurable rules — fare class mandates, preferred suppliers, advance booking windows, budget thresholds. Breaches trigger rejection or escalation.
Structured JSON Output
Normalised into a predefined schema — ready for direct consumption by downstream pricing services. Zero manual reformatting.
Smart Draft Generation
Composes quote response email or — when data is missing — a targeted clarification email asking only for the absent fields.
Adaptive Learning → Auto-send
Agent corrections become labelled training signals. Once confidence exceeds threshold per input pattern, that pattern graduates to auto-send. Kill switch reverts on confidence drop.
Additional Capabilities
Confidence Scoring per Field
Every AI-extracted field carries a score. Below threshold → flagged for agent review before any downstream action.
Quote Versioning
Full iteration history per request thread. Agents see what changed across clarification cycles.
Approval Workflow
High-value or out-of-policy quotes route through a configurable approval chain before the draft is sent.
SLA Tracking
Monitors response time per quote against agreed SLA windows. Escalates on breach.
Audit Trail & Logging
Every AI decision, policy check, extraction result, and draft action is logged — mandatory before auto-send is enabled on any pattern.
Multi-tenant Isolation
Per-tenant policy rules, pricing endpoints, email templates, output schemas, and confidence thresholds — fully isolated per client.
